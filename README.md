# marvel-heroes
Marvel heroes list made in React

### Ecosystem
- ReactJS
- Redux as Flux pattern
- Immutability with ImmutableJS
- Hot reloading
- Webpack
- Babel
- ES2015
- ESLint
- Karma
- PhantomJS
- Jasmine

### Installation
```bash
$ npm install
```

### Run locally
```bash
$ npm start
```

### Build
```bash
$ npm run build
```

### Run tests
```bash
$ npm run test
```
